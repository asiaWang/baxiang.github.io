####Mac10.9系统已经自带有Git功能了，但是在这里还是需要介绍以下怎样在Mac上安装Git。
	
1.首先我们打开网址www.github.com,在网站首页我们可以看到如图：

![Mou icon](https://raw.githubusercontent.com/MacMang/allImaga/master/setGit.png)

2.我们点击红色框框里面的set up Git,这时候会进入一个新的页面 <https://help.github.com/articles/set-up-git>
如图:红色文字标注处，点击下载和安装最新版本的GIT
<!--More-->

![Mou icon](https://raw.githubusercontent.com/MacMang/allImaga/master/DownloadGit.png)

3.这时候我们进到git官网 <http://www.git-scm.com/download> 我们选择相应的版本来下载（ [我的网盘](http://pan.baidu.com/s/1o6NVWeu)也提供下载最新版本的Git）。我选择了MacOSX.弹出以下提示

![Mou icon](https://raw.githubusercontent.com/MacMang/allImaga/master/autoDownload.png)

4.当Git成功下载之后，只需要正常的双击安装就可以了。

5.安装成功之后,我们打开终端输入 git - -version，这时候我们就可以看到自己Git的版本了。
